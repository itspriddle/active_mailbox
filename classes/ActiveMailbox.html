VOICEMAIL_ROOT = ENV['ASTERISK_VOICEMAIL_ROOT'] || "/var/spool/asterisk/voicemail"
 

The directory Asterisk records voicemail in

By default, this is `/var/spool/asterisk/voicemail`. If it isn’t, you did something fancy when building Asterisk. Set ENV[‘ASTERISK_VOICEMAIL_ROOT’] to that directory in your library or add ‘export ASTERISK_VOICEMAIL_ROOT=“/my/voicemail“’ to your ~/.bashrc
